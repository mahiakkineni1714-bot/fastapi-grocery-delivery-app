# Grocery Delivery FastAPI Project

## 📌 Description
This is a backend API project built using FastAPI.  
It simulates a grocery delivery system with products, cart, and orders.

## 🚀 Features
- GET APIs
- POST APIs with Pydantic validation
- CRUD operations
- Cart & Checkout workflow
- Search, Sort, Pagination
- Swagger API testing

## 🛠 Tech Stack
- Python
- FastAPI
- Uvicorn

## How to Run
1. Install dependencies:
   pip install fastapi uvicorn

2. Run server:
   uvicorn main:app --reload

3. Open:
   http://127.0.0.1:8000/docs
